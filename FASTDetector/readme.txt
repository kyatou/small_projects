#FAST Detector test

##Requirements
- OpenCV 2.3 or later
- Ubuntu or other linux distribution

##How to build
gcc fastdetector.cpp -o fast.o -lopencv_highgui -lopencv_core -lopencv_features2d -lopencv_imgproc

##How to RUN
- Connect WebCamera to your computer.
- run ./fast.o
- "esc" key to exit.

Have fun!!
